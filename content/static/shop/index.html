For the third year, Processing has partnered with Wire & Twine to sell t-shirts to fund software development. This fall, we're featuring a new design by Karsten Schmidt. This shirt is on sale only until 19 September 2011.
 
Style P52011A \ TOXI
 


The TOXI shirt was created with Karsten Schmidt's extensive Toxiclibs library for Processing in combination with LuxRender. You can see process images and alternate explorations for the shirt on Flickr.

There are two color options to choose from: metallic silver on navy and light blue on gray.
 
Style P52010A \ The BRAIN (DISCONTINUED)
 


The amazing Nervous System created the elegant image printed on the front of this shirt. The image is printed in white ink on a gray shirt.
 
 
Style P52009A \ The BEAUTY (DISCONTINUED)
 


This shirt features a network image created with the code written for the Processing logo. It's a three-color silkscreen image on navy.
 
 
Style P52010B \ The BEAST (DISCONTINUED)
 


Based on the 2009 BEAST tee, this shirt features a wireframe Processing IDE with source code on the front. The code, which generates the image on the back of the shirt, draws a line on the z-axis based on the brightness of the corresponding pixel. The graphics are white ink on a navy shirt.
